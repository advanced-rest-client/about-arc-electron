## &lt;about-arc-electron&gt;

About screen for Advanced REST Client - Electron release.

## Development

```sh
git clone https://github.com/advanced-rest-client/about-arc-electron
cd about-arc-electron
npm install
```

### Running the demo locally

```sh
npm start
```

### Running the tests

```sh
npm test
```
